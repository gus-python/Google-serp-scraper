Ferramenta de automação web para extrair os 10 primeiros resultados orgânicos do Google para qualquer keyword, salvando títulos e links em uma planilha
-Principais funcionalidades
* Solicita uma keyword ao usuário via terminal
* Abre o Chrome automaticamente
* Acessa o Google e digita a keyword na barra de busca
* Aguarda os resultados carregarem completamente
* Extrai os títulos e links dos 10 primeiros resultados orgânicos
* Salva tudo em um arquivo Google_top_10.csv
- Tecnologias utilizadas
* Python
* Selenium (Automação de navegador)
* CSV
- Lógica do projeto
  O script automatiza o processo de busca no Google da seguinte forma:
1. Solicita uma keyword ao usuário via terminal
2. Abre o Chrome automaticamente
3. Acessa o Google e digita a keyword na barra de busca
4. Aguarda os resultados carregarem completamente
5. Extrai os títulos e links dos 10 primeiros resultados orgânicos
6. Salva tudo em um arquivo Google_top_10.csv
